# Rick and Morty Explorer 🛸

Aplicação web que consome a API do Rick and Morty e exibe os personagens em cards interativos.

![API](https://github.com/user-attachments/assets/c6fed320-96cf-4452-9704-9e15ecf9be42)

## 🚀 Tecnologias

- TypeScript
- HTML/CSS
- Rick and Morty API

## ✨ Funcionalidades

- Listagem dos personagens em cards
- Status do personagem com indicador visual colorido (vivo, morto ou desconhecido)
- Exibe nome, espécie, status e última localização conhecida
- Busca de personagens por nome em tempo real

## ⚙️ Como rodar
```bash
npm install
npm run dev
```

Abra o `index.html` no navegador.

## 🔗 API utilizada

[Rick and Morty API](https://rickandmortyapi.com)
